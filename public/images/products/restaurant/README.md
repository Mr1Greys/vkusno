# Restaurant images

К папке `public/images/products/restaurant` добавлены директории для новых категорий меню:

- `salads`
- `snacks`
- `soups`
- `hot-dishes`

Добавляйте сюда файлы изображений для карточек ресторанного меню. Имена файлов можно использовать понятные, например:

- `crispy-eggplants.png`
- `caesar-chicken.png`
- `pilaf.png`

Если нужно, могу сразу подготовить структуру `imageUrl` в данных под эти папки.
