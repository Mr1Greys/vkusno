## Инструкции
1. Создайте соответствующие папки в `public/images/products/`
2. Загружайте изображения в формате PNG с именами, указанными выше
3. Размер изображений: рекомендуется 400x400 пикселей для квадратного отображения
4. После загрузки изображений перезапустите приложение для обновления кэша
